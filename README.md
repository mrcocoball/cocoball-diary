## 사용 스택

* Java
* Spring Boot
* Spring Security
* Spring Properties DevTools
* Spring Data JPA
* Spring Data REST
* Swagger UI
* Querydsl
* MySQL
* Thymeleaf
* JavaScript / Axios
* GitKraken
* AWS RDS / EC2 </br></br>
